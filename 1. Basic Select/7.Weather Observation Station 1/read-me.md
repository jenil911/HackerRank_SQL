# STATION Table Query

This repository contains information about querying a list of `CITY` and `STATE` from the `STATION` table.

## Table Description

The `STATION` table is described as follows:

| Field        | Type           |
|--------------|----------------|
| ID           | NUMBER         |
| CITY         | VARCHAR   |
| STATE        | VARCHAR  |
| LAT_N        | NUMBER         |
| LONG_W       | NUMBER         |
