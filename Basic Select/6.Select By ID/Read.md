# CITY Table Query

This repository contains information about querying all columns for a city in the `CITY` table with the ID `1661`.

## Table Description

The `CITY` table is described as follows:

| Field        | Type           |
|--------------|----------------|
| ID           | NUMBER         |
| NAME         | VARCHAR2(17)   |
| COUNTRYCODE  | VARCHAR2(3)    |
| DISTRICT     | VARCHAR2(20)   |
| POPULATION   | NUMBER         |
